# DopaMeter

    Анализ влияния экранного времени на сон и поведение пользователей

## Descrtiption
**DopaMeter** – это Python-проект для анализа цифрового поведения пользователей на основе данных об экранном времени, сне, работе и досуге.

## Data

Датасет содержит информацию о 400 пользователях:

 * экранное время (общее, рабочее, развлекательное)
 * длительность сна
 * возраст
 * формат работы (удаленный / гибрид / очный)
 * поведенчиские метрики

## Stack
 * Python
 * pandas
 * numpy
 * pytest
 * argparse

## How to use
    python main.py stats
    python main.py analyze
    python main.py report
    python main.py risk

## Generators
В проекте реализованы генераторы для поэлементного перебора пользователей и выборки пользователей с высоким уровнем риска. Они интегрированы в CLI через команды users и highrisk.

## Decorators
Используются декораторы:
 * cache_results – для кэширования результатов анализа
 * validate_columns – для провеки корректности данных
